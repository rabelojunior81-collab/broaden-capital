# Broaden Capital

> Public landing page / repository front door for developers, partners, and future GitHub Pages routing.

---

## PT-BR

**Broaden Capital** é uma landing page institucional criada para apresentar a gestora de ativos alternativos com linguagem visual mais sofisticada, glassmorphism, background dinâmico e seções que surgem conforme o scroll.

### O que tem aqui
- `index.html` — a landing page principal
- `styles.css` — identidade visual, glassmorphism, responsividade e camadas visuais
- `design-spec.md` — direção de design e premissas visuais

### Direção visual
- Fundo com profundidade, glows e partículas discretas
- Cards translúcidos, mas legíveis
- Header móvel com menu hamburguer funcional
- Reveal-on-scroll para uma experiência de página moderna

### Uso
Abra `index.html` no navegador ou publique o repositório via **GitHub Pages** apontando para a raiz do projeto.

### Observação
Este repositório foi pensado como **landing page alternativa para devs** também, então o README funciona como uma porta de entrada rápida para entender o projeto sem abrir o código primeiro.

## EN-US

**Broaden Capital** is a public institutional landing page designed to present an alternative-assets manager with a more premium visual language: glassmorphism, ambient background motion, and scroll-triggered reveals.

### What’s included
- `index.html` — main landing page
- `styles.css` — visual system, glass layers, responsiveness, and motion
- `design-spec.md` — design direction and visual constraints

### Visual direction
- Depth-rich background with subtle particles and glow
- Translucent but readable cards
- Functional mobile header and hamburger menu
- Reveal-on-scroll sections for a modern landing-page feel

### Usage
Open `index.html` locally or publish the repo through **GitHub Pages** using the repository root as the source.

### Note
This repo is meant to work as a **developer-facing landing page alternative** too, so the README acts like a lightweight product intro before anyone dives into the code.
